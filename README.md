# Single Login Lock

Restrict WordPress users to a single active login session across devices and browsers.

---

## 📖 Overview

**Single Login Lock** is a lightweight WordPress security plugin that prevents users from maintaining multiple active sessions simultaneously.

When a user logs in from a new device or browser, the plugin automatically invalidates the previous session, ensuring that only one active login exists at any given time. This helps reduce account sharing, improve security, and maintain better control over user access.

---

## ✨ Features

* Enforce a single active login session per user
* Automatically terminate previous sessions on new login
* Device and browser session protection
* Prevent unauthorized account sharing
* Improve website security and access control
* Lightweight and easy to use
* No complex configuration required

---

## 🚀 How It Works

1. A user logs into your WordPress website.
2. The user later logs in from another device or browser.
3. The new login becomes active.
4. The previous session is automatically terminated.
5. Only one active session remains.

## 🔧 Installation

1. Download the plugin ZIP file.
2. Navigate to **WordPress Admin → Plugins → Add New**.
3. Click **Upload Plugin**.
4. Select the ZIP file and install.
5. Activate the plugin.
6. The plugin will automatically begin enforcing single-session logins.
